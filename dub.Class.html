C++ Class definition.

(internal) A C++ class/struct definition.

.__call (lib, self)

This lets sub-classes of dub.Class like dub.Namespace or dub.CTemplate use the same call convention to create new objects.

.new (def)

Create a new class definition. Most of the attributes are passed with def. Usage example:

local class = dub.Class {
  -- self can be a class or db (root)
  db      = self.db or self,
  parent  = parent,
  name    = name,
  xml     = elem,
  xml_headers  = {
    {path = header.dir .. lub.Dir.sep .. elem.refid .. '.xml', dir = header.dir}
  },
}

:findChild (name)

Return a child element from name or nil if nothing is found. Uses the database internally.

Return a method from a given @name@.
