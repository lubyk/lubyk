Lubyk documentation

  • lut

Lubyk utility module

Currently, this module contains a documentation generator and testing facilities.

This module is part of the lubyk project. MIT license © Gaspard Bucher 2014.

Installation

With luarocks:

$ luarocks install lut

With luadist:

$ luadist install lut

.VERSION = '1.0.3'

Current version of 'lut' respecting semantic versioning.

Library dependencies

.DEPENDS = {

'lua >= 5.1, < 5.3'

Compatible with Lua 5.1, 5.2 and LuaJIT

'lub >= 1.0.3, < 1.1'

Uses Lubyk base library

}

Doc

lut.Doc parses lua comments and extracts the documentation from these comments.

Test

The test suite is used to declare tests and group them by class or module.
